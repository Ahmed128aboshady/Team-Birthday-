# لوحة أعياد ميلاد المكتب التفاعلية | Office Birthdays Tracker

لوحة أعياد ميلاد تفاعلية ممتازة واحترافية للمكتب، صممت خصيصاً لمتابعة مناسبات 16 زميلاً في فريق العمل باللغتين العربية والإنجليزية.

تتميز اللوحة بتصميم زجاجي عصري (Glassmorphism) مع دعم كامل للوضع الداكن (Dark Mode)، والعد التنازلي التلقائي لأيام أعياد الميلاد المتبقية، وحساب الأعمار والأبراج الفلكية تلقائياً، والاحتفال بالقصاصات الملونة (Confetti).

---

## ✨ المميزات الرئيسية

1. **دعم كامل للغتين (العربية/الانجليزية):** تغيير لغة الواجهة واتجاهها بالكامل بنقرة واحدة (RTL/LTR).
2. **شريط البحث والتصفية الذكي:**
   - البحث بالاسم أو القسم.
   - فلترة الموظفين حسب شهر ميلادهم.
   - ترتيب القائمة حسب الأقرب عيد ميلاد، أبجدياً، أو حسب السن.
3. **التنبيه بأعياد ميلاد اليوم:** يظهر بانر احتفالي مميز ومتحرك عند مصادفة عيد ميلاد أحد الزملاء اليوم مع إمكانية إطلاق ألعاب نارية/confetti.
4. **تعديل البيانات مباشرة:** إمكانية إضافة موظفين جدد، تعديل بيانات الموظفين الحاليين، أو حذفهم مباشرة من الواجهة.
5. **تصدير واستيراد البيانات (Export/Import):** حفظ نسخة احتياطية من قائمة أعياد الميلاد كملف `members.json` واسترجاعها في أي وقت.
6. **احصائيات ذكية:** رسم بياني يوضح توزيع أعياد الميلاد على مدار شهور السنة الـ 12.

## 🚀 طريقة التشغيل محلياً (Run Locally)

المشروع عبارة عن صفحة ويب ثابتة (Static SPA) تعمل مباشرة على أي متصفح.
لتجنب مشاكل المتصفح الأمنية عند قراءة ملف `members.json` محلياً (CORS Policy)، يفضل تشغيل خادم محلي خفيف كالتالي:

### الخيار الأول: باستخدام Python (الأسهل)
إذا كان لديك Python مثبتاً على جهازك، افتح Terminal في مجلد المشروع ونفذ:
```bash
python -m http.server 8000
```
ثم افتح المتصفح واذهب إلى: `http://localhost:8000`

### الخيار الثاني: باستخدام Node.js
إذا كان لديك Node.js:
```bash
# تثبيت خادم محلي سريع
npm install -g serve
# تشغيل الخادم
serve .
```
أو ببساطة افتح ملف `index.html` في متصفحك (ملاحظة: في هذه الحالة سيتم تحميل البيانات الافتراضية من الذاكرة الاحتياطية مباشرة إذا حظر المتصفح قراءة ملف الـ JSON).

## 🛠️ كيف تقوم برفع المشروع على GitHub وتفعيله مجاناً؟

بما أن المشروع لا يحتوي على خوادم خلفية (Serverless)، يمكنك استضافته مجاناً بالكامل على **GitHub Pages** ليتمكن زملائك في العمل من تصفحه في أي وقت!

### الخطوة 1: تهيئة Git ورفع المشروع
1. قم بإنشاء مستودع جديد (New Repository) على حسابك في GitHub وسمه مثلاً `office-birthdays`.
2. افتح سطر الأوامر (Terminal/PowerShell) في مجلد المشروع على جهازك ونفذ الأوامر التالية:
   ```bash
   git init
   git add .
   git commit -m "Initial commit - Birthday Tracker"
   git branch -M main
   git remote add origin https://github.com/اسم_حسابك/اسم_المستودع.git
   git push -u origin main
   ```
   *(قم باستبدال `اسم_حسابك` و `اسم_المستودع` بروابطك الفعلية).*

### الخطوة 2: تفعيل الاستضافة (GitHub Pages)
1. ادخل إلى صفحة المستودع الخاص بك على GitHub.
2. اذهب إلى التبويب **Settings** (الإعدادات).
3. من القائمة الجانبية اختر **Pages**.
4. تحت قسم **Build and deployment**، اختر الفرع `main` (أو `root`) ثم اضغط **Save**.
5. انتظر دقيقة واحدة، وستظهر لك رسالة بالأعلى تحتوي على رابط الموقع المباشر (مثل: `https://username.github.io/office-birthdays`).

## 📂 هيكلية ملفات المشروع
* `index.html`: هيكل الموقع الرئيسي واللوحة التفاعلية والنافذة المنبثقة.
* `style.css`: التصميم الأنيق وتوزيع الألوان والمؤثرات البصرية.
* `app.js`: محرك التطبيق، العمليات الحسابية للأبراج والأعمار، وإطلاق الاحتفالات.
* `members.json`: قاعدة البيانات الافتراضية للفريق (تضم 16 شخصاً افتراضياً جاهزين للتعديل).
